# MCP Academy: The Three Magic Paths

An interactive story-driven game for kids to learn MCP (Model Context Protocol) fundamentals.

## What kids will learn
- What Tools are (things that DO)
- What Resources are (things that KNOW)
- What Prompts are (things that GUIDE)

## How to run
1. Install dependencies
2. Run dev server
3. Explore the academy

## Goal
Make learning system design intuitive and fun.
